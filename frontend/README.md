## Setup

1. Install dependencies:
```bash
npm install
```

2. Set environment variables:
```bash
# Create .env file
VITE_API_URL=https://api.buildforu.pw
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
